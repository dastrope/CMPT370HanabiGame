5.2.4 Constraints

Full functionality of the application will be restrained by a required connection to a UofS server used for hosting games of Hanabi.

	
5.2.5 Assumptions and Dependencies

The application will be implemented using the Java Development Kit and will be exported as a .Jar file. Any Windows, Mac, or Linux OS running the latest version of Java will execute as intended.
